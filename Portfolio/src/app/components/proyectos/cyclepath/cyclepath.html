# ¿Qué es CyclePath?

CyclePath es un mapa interactivo que muestra la red ciclista de cualquier ciudad del mundo. Busca una ciudad y visualiza al instante sus carriles bici clasificados por tipo: vías exclusivas, carriles en calzada y aceras bici. 

// Datos en tiempo real desde OpenStreetMap vía Overpass API

Además del mapa, incluye un calculador de rutas punto a punto que usa el algoritmo A* para trazar el camino más corto utilizando exclusivamente infraestructura ciclista disponible.

# Funcionalidades

Búsqueda global

Busca cualquier ciudad del mundo con autocompletado en tiempo real.

Mapa interactivo

Red ciclista superpuesta sobre el mapa base, con zoom y desplazamiento fluido.

Tipos de carril

Diferencia entre vías exclusivas, carriles en calzada y aceras bici por color.

Calculador de rutas

Rutas punto a punto usando A*, exclusivamente por infraestructura ciclista.

Datos en tiempo real

Los carriles se consultan al momento desde OpenStreetMap, siempre actualizados.

Sin dependencias de build

JavaScript, HTML y CSS vanilla. Sin frameworks, sin bundler, carga instantánea.

# Tipos de carril

Vía ciclista exclusiva Carretera o camino reservado únicamente para bicicletas
Carril en calzada Carril bici señalizado dentro de la vía de tráfico rodado
Acera bici / otro Vía compartida con peatones u otro tipo de infraestructura ciclista

# Tecnologías usadas

Leaflet Mapas interactivos en el navegador
Overpass API Consulta de datos ciclistas desde OpenStreetMap
Nominatim Geocodificación y autocompletado de ciudades
A* Algoritmo de búsqueda de rutas sobre el grafo ciclista
JS / HTML / CSS Vanilla, sin frameworks ni bundler
